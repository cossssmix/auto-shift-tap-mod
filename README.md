# Shift-Tap Mod (Minecraft 1.21.11 / Fabric)

Простой легитный мод для PvP.

* **Shift-Tap**: автоматическое приседание (Shift-Tap) при ударах по игрокам/мобам. Не работает в воде, лаве и паутине.
* **AutoSprint**: легитный автоспринт через игровые настройки.

## Управление
* Чтобы открыть меню настроек (ClickGui), **его необходимо забиндить в стандартных настройках управления Minecraft** (по умолчанию клавиша не назначена, категория «Shift Tap»).
