# Shift-Tap Mod (Minecraft 1.21.11 / Fabric)

Простой легитный клиентский мод для PvP, реализующий функцию автоматического Shift-Tap (приседания) при ударах по сущностям, а также легитный AutoSprint. Все чит-функции полностью удалены.

## Возможности

### 1. Shift-Tap Module
При нанесении удара по живой сущности (игроку или мобу) мод автоматически зажимает клавишу приседания (Sneak/Shift) на заданное количество тиков.
* **Настройка тиков**: в меню можно задать длительность зажатия (от 1 до 20 тиков, по умолчанию 5).
* **Условия срабатывания**:
  * **Only In Air**: срабатывать только когда вы находитесь в воздухе.
  * **Only On Ground**: срабатывать только когда вы находитесь на земле.
* **Встроенные ограничения**: мод автоматически **не срабатывает**, если вы находитесь в воде, лаве или паутине.

### 2. AutoSprint Module
Легитный автоспринт, который каждый тик зажимает клавишу спринта через стандартные настройки (`sprintKey.setPressed(true)`). Безопасен для использования и не вызывает подозрений у античитов.

### 3. Автосохранение настроек
Все настройки модулей сохраняются автоматически при их изменении. Конфигурационные файлы создаются в папке `.minecraft/config/`:
* `shifttap.json` — конфигурация модуля Shift-Tap.
* `autosprint.json` — конфигурация модуля AutoSprint.

---

## Управление и интерфейс

* **`J`** — Открыть меню настроек (ClickGui).
* **`V`** — Быстрое включение/выключение модуля Shift-Tap.
* **`G`** — Быстрое включение/выключение модуля AutoSprint.

---

## Сборка проекта

Для сборки мода выполните в терминале в корневой папке проекта:

```bash
# Windows
.\gradlew.bat build

# Linux / macOS
./gradlew build
```

Готовый файл `.jar` будет находиться по пути `build/libs/` с именем формата:
`shifttap-fabric-<версия-мода>+mc1.21.11.jar`.
